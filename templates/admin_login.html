Admin Access

JWT Authentication Required

Authentication Method:
This endpoint requires a valid JWT token with admin privileges.

Set token via:
  • Cookie: auth_token=YOUR_JWT
  • Header: Authorization: Bearer YOUR_JWT
Hint:
JWT tokens are signed with a secret key. If you can find the key, you can forge your own token.

Required Claims: 

{
  "role": "admin",
  "username": "admin"
}
